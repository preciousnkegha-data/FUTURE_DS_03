# FUTURE_DS_03
A compact, end-to-end survey analysis project demonstrating data cleaning, visualization, and sentiment extraction using Python. Includes a professional report summarizing satisfaction insights, key findings, and recommendations.


 ## Student Satisfaction Survey Analysis Project

A Python-based analysis of student course feedback with visual insights and sentiment-driven findings.

## Overview

This project analyzes course-based survey responses to provide actionable insights into student satisfaction and institutional performance.
It covers data cleaning, exploratory analysis, visualization, sentiment extraction, and a professional report summarizing key findings and recommendations.

## Objectives

* Clean and prepare raw survey data for analysis
* Extract and benchmark satisfaction scores
* Explore trends across courses
* Apply sentiment analysis on survey text
* Visualize insights using Seaborn and Matplotlib
* Produce a professional PDF report and executive summary

## Tools & Technologies

* Python
* Pandas, NumPy
* Seaborn, Matplotlib
* TextBlob (NLP & sentiment analysis)
* WordCloud
* Jupyter Notebook

## Key Insights

* Consistent participation across courses ensures reliable results
* Satisfaction baseline is 3.84/5.0, good but below excellence target
* Critical low scores (1.0) highlight areas requiring urgent attention
* Significant performance variation between courses
* Instructional quality, syllabus coverage, and technology integration are primary drivers of satisfaction
* NLP and WordCloud analyses reveal common concerns and priorities

* Demonstrates end-to-end data cleaning and preparation
* Produces visual and thematic insights
* Applies sentiment analysis to uncover student priorities
